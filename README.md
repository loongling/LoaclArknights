# ArknightsServer
这是一个明日方舟本地服务端,对应的客户端是CN 2.4.61,该服务端理论上支持多人游戏,但仅仅支持单IP登录，如果多人在同一公网IP下登录可能会覆盖其他人的secret导致他人同步SyncData时读取的最新登录人的账户数据
## 怎么启动/解决中文乱码问题
请使用java17以上的java版本运行

1、首先运行 `chcp 65001`  
2、在项目根目录运行`java -jar hypergryph-1.9.3.jar`
