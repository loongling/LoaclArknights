# ArknightsServer
这是一个明日方舟本地服务端,对应的客户端是CN 2.4.61,该服务端理论上支持多人游戏,但可能因为登陆人在同一局域网访问公网服务器可能因为IP相同而造成串号的情况
## 怎么启动/解决中文乱码问题
请使用java17以上的java版本运行

1、首先运行 `chcp 65001`
2、在项目根目录运行`java -jar hypergryph-1.9.3.jar`
